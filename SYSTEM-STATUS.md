## 🔗 System Architecture (All Connected)

```

    ↓

    ↓

    ↓

```

### Connection Flow:

2. **Requests products** → api/products.php queries database
3. **Adds to cart** → api/cart.php manages session
4. **Places order** → api/orders.php saves to database
